# accesskey :#
规定激活元素的快捷键

### 浏览器支持 ###
几乎所有浏览器均支持accesskey属性，除了Opera

###提示和注释###
以下元素支持accesskey:`<a>, <area>, <button>, <input>, <label>, <legend>` 以及 `<textarea>`。
